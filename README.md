# Hair Care Landing

Промо-лендинг по подбору ухода за волосами для «Золотого Яблока».

## Стек

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Запуск локально

```bash
npm install
npm run dev
```

Приложение поднимется на `http://localhost:8080`.

## Сборка

```bash
npm run build
npm run preview
```

## Тесты

```bash
npm run test
```

## Структура

- `src/pages/Index.tsx` — главная страница лендинга
- `src/components/*Section.tsx` — секции страницы
- `src/index.css` — глобальная тема и стили
